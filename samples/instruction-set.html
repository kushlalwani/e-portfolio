Writing Sample

Instruction Set

This instruction set provides a complete, step-by-step guide for making fried rice in a home kitchen. It includes required equipment, ingredients, preparation steps, and cooking instructions, along with safety notes and a glossary of key terms.

The document is designed for beginners, emphasizing clear sequencing, visual organization, and practical usability. The process is broken into stages such as preparation, cooking, and final assembly so that users can follow along easily.

Reflection

This project helped me understand how to design instructions for usability rather than just completeness. I focused on organizing the process into clear stages, using consistent formatting, and anticipating where a user might get confused. I also learned the importance of defining terms like chop or to taste, which may not be obvious to beginners. This piece demonstrates my ability to create structured, user-centered documentation that prioritizes clarity, efficiency, and real-world application.
