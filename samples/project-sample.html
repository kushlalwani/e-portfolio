Writing Sample

Document Redesign

This memo analyzes the redesign of the HackPSU teardown plan, a document used by organizers to coordinate post-event operations. The redesign focuses on improving usability in a fast-paced environment through a clear workflow, standardized checklists, and better organization of information.

Key additions include a Quick Start Guide, improved task sequencing, and integrated instructions for inventory tracking. The redesigned document is intended to support real-time execution rather than simply present information.

Reflection

This project helped me understand how document design directly impacts usability, especially in high-pressure environments. I learned how to identify issues in an existing document and redesign it with the user in mind, focusing on clarity, structure, and efficiency. One of the biggest takeaways was the importance of guiding users through a process rather than simply listing information. This piece demonstrates my ability to think critically about communication design and create documents that support real-world tasks and collaboration.
